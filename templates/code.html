ตัวอย่างโค้ดและคำอธิบายการทำงาน

1. การสร้างแอป Flask

from flask import Flask
app = Flask(__name__)

โค้ดส่วนนี้ใช้สำหรับสร้างแอปพลิเคชันเว็บด้วย Flask ซึ่งเป็น Framework ของภาษา Python ที่ช่วยพัฒนาเว็บได้อย่างง่ายและรวดเร็ว โดย Flask จะทำหน้าที่จัดการเส้นทาง URL และการแสดงผลหน้าเว็บ

2. การสร้าง Route

@app.route("/")
def home():
    return render_template("index.html")

Route คือการกำหนดเส้นทางของ URL เมื่อผู้ใช้งานเข้าเว็บไซต์ที่หน้าแรก ("/") ระบบจะเรียกไฟล์ index.html มาแสดงผล ทำให้หน้าเว็บสามารถแบ่งเป็นหลายหน้าได้ภายในระบบเดียวกัน

3. การเชื่อมต่อไฟล์ HTML

return render_template("menu.html")

ฟังก์ชัน render_template ใช้เรียกไฟล์ HTML ที่อยู่ในโฟลเดอร์ templates เพื่อแสดงผลหน้าเว็บตามที่กำหนด ทำให้สามารถแยกโครงสร้างหน้าเว็บออกจากโค้ด Python ได้อย่างเป็นระบบ

4. การกำหนด Host และ Port สำหรับ Deploy

if __name__ == "__main__":
    app.run(host="0.0.0.0", port=10000)

คำสั่งนี้ใช้กำหนดให้แอปพลิเคชันทำงานบนเซิร์ฟเวอร์ของ Render โดย host="0.0.0.0" เปิดให้เข้าถึงจากภายนอก และ port=10000 เป็นพอร์ตที่ Render กำหนดให้ใช้งาน
